# Maintenance Report Generation

Extract every maintenance event.

Generate structured maintenance history.

Each event should include:

- Date
- Engineer
- Maintenance Type
- Issue
- Symptoms
- Root Cause
- Action Taken
- Parts Replaced
- Downtime
- Recommendation
- Next Maintenance Date

If dates are missing,

estimate chronology from context.

Generate a complete maintenance history.

Save as:

maintenance_report.md
